About Ravi Kant

This game was created by Ravi Kant, a BTech student at Lovely Professional University in Punjab. Ravi is passionate about game development and aims to bring joy through interactive entertainment.

Technologies Used:

  • HTML: For structuring the web pages.
  • CSS: For styling the elements and ensuring a responsive design.
  • JavaScript: To add interactivity and game logic.
  • Audio: To enhance the user experience with sound effects.
  • Video: For a dynamic background effect.

Game Algorithm:

The game uses a simple algorithm to handle the game logic. It checks for winning conditions after each move and determines if the game is won, lost, or tied. The computer player uses a basic random move selection algorithm in the normal mode, with potential for more advanced AI logic in higher difficulty settings.
